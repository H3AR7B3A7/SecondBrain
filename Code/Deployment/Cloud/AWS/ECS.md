# Elastic Container Service

A container orchestration Service that supports Docker & Windows Containers.

Similar to Kubernetes, but with deep integration with AWS services
- IAM
- VPC
- Route53

Options
- Cluster of VMs
- Fargate for Serverless
- EC2 for more control

Elastic Container Registry is where you store your container images
ECS uses the images to deploy the container

Services that use ECS
- Sagemaker
- Lex
- amazon.com
