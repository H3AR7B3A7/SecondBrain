# Amplify

Advantages
- Frontend developers can focus on the web frontend
- Takes care creating a reliable backend
- Integrates you application with services
	- Cognito
	- S3
	- Lambda
	- API Gateway


- Hosting
	- Fully managed web hosting services
	- Web applications and static website hosting
	- CI/CD funtionality
	- Integrates with your code repository
- Studio
	- Simple visual interface, separate from AWS Management Console
	- Create your frontend UI
	- Configure and maintain your application backend
	- Add features like authentication, integration with data services, and serverless functions
