# Commands
## Why use commands?
- To have greater control
- To be faster
- For automation
- To use headless clients
- For cloud computing
- ...


## Most popular
Manual
> man "command"

Print Working Directory
> pwd

Print user id
> whoami

Show who is logged in
> who

List
> ls

???
> ps auxww

Clear terminal
> clear
